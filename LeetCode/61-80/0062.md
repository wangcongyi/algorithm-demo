```
Unique Paths
  一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？


```

#### A

```js
function Unique(m, n) {

    if (m <= 0 || n <= 0) return 0

    const dp = Array(m).fill(null).map(() => Array(n).fill(null))

    for (let i = 0; i < m; i++) {
      dp[i][0] = 1
    }

    for (let j = 0; j < n; j++) {
      dp[0][j] = 1
    }

    for (let x = 1; x < m; x++) {
      for (let y = 1; y < n; y++) {
        dp[x][y] = dp[x - 1][y] + dp[x][y - 1]
      }
    }
    console.log(dp)
    return dp[m - 1][n - 1]
  }
```
