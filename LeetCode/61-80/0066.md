#### Q

```

给定一个由 整数 组成的 非空 数组所表示的非负整数，在该数的基础上加一。
最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。
你可以假设除了整数 0 之外，这个整数不会以零开头。

```

#### A

```js

const plusOne = function (digits) {
  const length = digits.length
  let i = length - 1
  digits[i]++
  while (i > 0 && digits[i] > 9) {
    digits[i] = 0
    digits[i - 1]++
    i--
  }
  if (i === 0 && digits[0] > 9) {
    digits[0] = 0
    digits.unshift(1)
  }
  return digits
}

```
