#### Q

```

盛最多水的容器  

给你 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 
在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0) 
找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

```

#### A

```js

const maxArea = h => {
  let [i, j] = [0, h.length - 1]
  let max = 0

  while (i < j) {
    const min = Math.min(h[i], h[j])
    max = Math.max(max, min * (j - i))
    if (h[i] < h[j]) {
      i++
    } else {
      j--
    }
  }
  return max
}


```
