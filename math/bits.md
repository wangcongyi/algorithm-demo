### getBit

```js

const getBit = (number, bitPosition = 0) => {
  return (number >> bitPosition) & 1
}

```


```
右移运算符（ >> ）表示将一个数的二进制值向右移动指定的位数  
如果是正数，头部全部补 0；如果是负数，头部全部补 1   
右移运算符基本上相当于除以2的指定次方（最高位即符号位参与移动）  

// 10 >> 1   
// 5
因为 10 的 32 位二进制表示为 00000000000000000000000000001010  
右移 1 位                   00000000000000000000000000000101
即为 十进制的 5

右移运算可以模拟 2 的整除运算

5 >> 1   // 相当于 5 / 2 = 2  
21 >> 2  // 相当于 21 / 4 = 5
21 >> 3  // 相当于 21 / 8 = 2
21 >> 4  // 相当于 21 / 16 = 1

```
